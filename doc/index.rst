.. SiteDB documentation master file

CMS SiteDB
==========

The CMS SiteDB manages information about sites and people involved in the experiment,
with their associations and privileges for CMS data and workload management services.

This page documents the SiteDB v2 API only. If you are looking for
instructions on how to migrate from the v1 API, see
`SiteDBMigrationToV2 <https://twiki.cern.ch/twiki/bin/viewauth/CMSPublic/CompOpsSiteDBMigrationToV2>`_.
